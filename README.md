# telegram-appointment-bot
A simple Telegram Appointment Booking Bot built with Python.

## Features

✅ Book appointments through Telegram

✅ Collect user name

✅ Collect phone number

✅ Collect appointment date

✅ Send booking details to admin

✅ Easy to customize

## Commands

### /start

Start the bot.

### /book

Begin the appointment booking process.

### /cancel

Cancel the current booking.

## Installation

1. Clone the repository

```bash
git clone https://github.com/kawshl-beep/telegram-appointment-bot.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Add your bot token and admin ID

```python
TOKEN = "YOUR_BOT_TOKEN"
ADMIN_ID = YOUR_ADMIN_ID
```

4. Run the bot

```bash
python appointment_bot.py
```
